Local Network Media Distribution System (Laravel)

MediaLAN is a Laravel-based local network media distribution system that allows users to browse and download movies or TV series over a LAN (Local Area Network) using PIN-based access control. Each PIN has a limited number of downloads, making MediaLAN ideal for offline media sharing and monetization in shops, schools, or local communities.
